# 🚀 စတင်လေ့လာရန်အတွက် Cybersecurity – သင်ခန်းစာများ

AI နည်းပညာကို အလျင်အမြန် အသုံးချနေသော ယနေ့ခေတ်တွင် IT စနစ်များကို လုံခြုံစေရန် နားလည်မှုရှိခြင်းမှာ အရေးကြီးလွန်းသည်။ ဒီသင်တန်းကို သင်၏ လုံခြုံရေးလေ့လာမှုကို စတင်ရန် အခြေခံ Cybersecurity အယူအဆများကို သင်ပေးရန် ရည်ရွယ်ထားသည်။ Vendor များနှင့် မပတ်သက်ဘဲ သင်ခန်းစာတိုင်းကို ၃၀-၆၀ မိနစ်အတွင်း ပြီးမြောက်စေရန် ရည်ရွယ်ထားပြီး သင်ခန်းစာတိုင်းတွင် မေးခွန်းတိုတစ်ခုနှင့် နောက်ထပ်လေ့လာရန် လင့်ခ်များပါဝင်သည်။

![Cybersecurity for Beginners](../../translated_images/banner.cc5b05d7e5deed065123ba68678b48cbbfe411cb264c09cec64f58eda064a28a.my.jpg)

ဒီသင်တန်းမှာ ဘာတွေကို လေ့လာရမလဲ 📚  

- 🔐 CIA triad, risks, threats စသည်တို့အပါအဝင် အခြေခံ Cybersecurity အယူအဆများ  
- 🛡️ Security control ဆိုတာဘာလဲ၊ ဘယ်လိုပုံစံတွေရှိလဲဆိုတာ နားလည်ခြင်း  
- 🌐 Zero trust ဆိုတာဘာလဲ၊ ယနေ့ခေတ် Cybersecurity မှာ ဘာကြောင့် အရေးကြီးလဲဆိုတာ နားလည်ခြင်း  
- 🔑 Identity, networking, security operations, infrastructure, data security စသည်တို့နှင့် ပတ်သက်သော အဓိကအယူအဆများကို နားလည်ခြင်း  
- 🔧 Security control များကို အကောင်အထည်ဖော်ရန် အသုံးပြုသော tools များ၏ ဥပမာများ  

ဒီသင်တန်းမှာ မလေ့လာရမည့်အရာများ 🙅‍♂️  

- 🚫 Security tools အထူးသုံးနည်းများ  
- 🚫 "Hack" လုပ်နည်းများ သို့မဟုတ် Red teaming/offensive security  
- 🚫 Compliance standards အထူးလေ့လာခြင်း  

[![Watch the video](../../translated_images/intro_placeholder.f42382df518f233a1ea3cb1c82ae8f92732bc3ac4ac2b3138cb561d24ca91df5.my.png)](https://learn-video.azurefd.net/vod/player?id=a0fe1cef-c064-4d59-97a9-e89e12a99b4d)  

ဒီသင်တန်းကို ပြီးမြောက်ပြီးနောက် Microsoft Learn modules များကို ဆက်လက်လေ့လာနိုင်ပါသည်။ [Microsoft Security, Compliance, and Identity Fundamentals](https://learn.microsoft.com/training/paths/describe-concepts-of-security-compliance-identity/?WT.mc_id=academic-96948-sayoung) ကို ဆက်လက်လေ့လာရန် အကြံပြုပါသည်။  

နောက်ဆုံးတွင် [Exam SC-900: Microsoft Security, Compliance, and Identity Fundamentals exam](https://learn.microsoft.com/credentials/certifications/exams/sc-900/?WT.mc_id=academic-96948-sayoung) ကို စဉ်းစားနိုင်ပါသည်။  

> 💁 ဒီသင်တန်းနှင့် ပျောက်နေသော အကြောင်းအရာများအတွက် သင့်အကြံပြုချက်များကို ကျွန်ုပ်တို့အား မျှဝေပါ။
