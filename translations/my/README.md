# 🚀 Cybersecurity for Beginners – သင်ရိုးညွှန်းတမ်း

AI နည်းပညာအသုံးပြုမှု အရှိန်မြင့်လာသော ယနေ့ခေတ်တွင် IT စနစ်များကို ဘေးကင်းစွာ ထိန်းသိမ်းနိုင်ရန် နားလည်မှုမှာ အလွန်အရေးကြီးပါသည်။ ဒီသင်တန်းကို သင့်လျော်သော အခြေခံ Cybersecurity အကြောင်းအရာများကို သင်ကြားပေးရန် ရည်ရွယ်ထားပြီး သင့်လျော်သော လေ့လာမှုကို စတင်ရန် အထောက်အကူဖြစ်စေပါသည်။ Vendor အခြေခံမဟုတ်ဘဲ 30-60 မိနစ်အတွင်း ပြီးမြောက်နိုင်သော သေးငယ်သော သင်ခန်းစာများအဖြစ် ခွဲခြားထားပါသည်။ သင်ခန်းစာတိုင်းတွင် မေးခွန်းတိုတစ်ခုနှင့် နောက်ထပ်လေ့လာရန် လင့်များပါဝင်ပါသည်။

![Cybersecurity for Beginners](../../translated_images/banner.cc5b05d7e5deed065123ba68678b48cbbfe411cb264c09cec64f58eda064a28a.my.jpg)

ဒီသင်တန်းတွင် ပါဝင်သောအရာများ 📚

- 🔐 CIA triad, risks, threats စသည်တို့ကဲ့သို့သော အခြေခံ Cybersecurity အကြောင်းအရာများ
- 🛡️ Security control ဆိုတာဘာလဲ၊ ဘယ်လိုပုံစံတွေရှိလဲဆိုတာ နားလည်ခြင်း
- 🌐 Zero trust ဆိုတာဘာလဲ၊ ယနေ့ခေတ် Cybersecurity တွင် ဘာကြောင့် အရေးကြီးလဲဆိုတာ နားလည်ခြင်း
- 🔑 Identity, networking, security operations, infrastructure, data security စသည်တို့နှင့် ပတ်သက်သော အဓိကအကြောင်းအရာများကို နားလည်ခြင်း
- 🔧 Security controls များကို အကောင်အထည်ဖော်ရန် အသုံးပြုသော ကိရိယာများ၏ ဥပမာများ

ဒီသင်တန်းတွင် မပါဝင်သောအရာများ 🙅‍♂️

- 🚫 သီးခြားသော Security tools များကို ဘယ်လိုအသုံးပြုရမည်ဆိုတာ
- 🚫 "Hack" လုပ်ခြင်း သို့မဟုတ် Red teaming/offensive security
- 🚫 သီးခြားသော Compliance standards များကို လေ့လာခြင်း

[![Watch the video](../../translated_images/intro_placeholder.f42382df518f233a1ea3cb1c82ae8f92732bc3ac4ac2b3138cb561d24ca91df5.my.png)](https://learn-video.azurefd.net/vod/player?id=a0fe1cef-c064-4d59-97a9-e89e12a99b4d)

ဒီသင်တန်းကို ပြီးဆုံးပြီးပါက Microsoft Learn modules များကို ဆက်လက်လေ့လာနိုင်ပါသည်။ [Microsoft Security, Compliance, and Identity Fundamentals](https://learn.microsoft.com/training/paths/describe-concepts-of-security-compliance-identity/?WT.mc_id=academic-96948-sayoung) ကို ဆက်လက်လေ့လာရန် အကြံပြုပါသည်။

နောက်ဆုံးတွင် [Exam SC-900: Microsoft Security, Compliance, and Identity Fundamentals exam](https://learn.microsoft.com/credentials/certifications/exams/sc-900/?WT.mc_id=academic-96948-sayoung) ကို စဉ်းစားနိုင်ပါသည်။

> 💁 ဒီသင်တန်းနှင့် ပတ်သက်သော အကြံပြုချက်များ သို့မဟုတ် ကျွန်ုပ်တို့ မပါဝင်ထားသော အကြောင်းအရာများအတွက် သင့်အကြံကို ကြားရရန် စိတ်အားထက်သန်ပါသည်။
