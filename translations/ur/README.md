# 🚀 ابتدائی افراد کے لیے سائبر سیکیورٹی – ایک نصاب  

AI ٹیکنالوجی کے اپنانے کے اس تیزی سے بدلتے دور میں، IT سسٹمز کو محفوظ بنانے کی اہمیت پہلے سے کہیں زیادہ ہے۔ یہ کورس آپ کو بنیادی سائبر سیکیورٹی تصورات سکھانے کے لیے ڈیزائن کیا گیا ہے تاکہ آپ اپنی سیکیورٹی کی تعلیم کا آغاز کر سکیں۔ یہ کورس کسی خاص وینڈر سے وابستہ نہیں ہے اور چھوٹے اسباق میں تقسیم کیا گیا ہے جنہیں مکمل کرنے میں تقریباً 30-60 منٹ لگتے ہیں۔ ہر سبق میں ایک چھوٹا سا کوئز اور مزید مطالعہ کے لنکس شامل ہیں اگر آپ موضوع کو مزید گہرائی میں سمجھنا چاہتے ہیں۔  

![ابتدائی افراد کے لیے سائبر سیکیورٹی](../../translated_images/banner.cc5b05d7e5deed065123ba68678b48cbbfe411cb264c09cec64f58eda064a28a.ur.jpg)  

یہ کورس کیا شامل کرتا ہے 📚  

- 🔐 بنیادی سائبر سیکیورٹی تصورات جیسے CIA ٹرائیڈ، خطرات اور خطرات کے درمیان فرق وغیرہ۔  
- 🛡️ سیکیورٹی کنٹرول کیا ہے اور یہ کس شکل میں آتے ہیں، سمجھنا۔  
- 🌐 زیرو ٹرسٹ کیا ہے اور جدید سائبر سیکیورٹی میں اس کی اہمیت کیوں ہے، سمجھنا۔  
- 🔑 شناخت، نیٹ ورکنگ، سیکیورٹی آپریشنز، انفراسٹرکچر اور ڈیٹا سیکیورٹی کے کلیدی تصورات اور موضوعات کو سمجھنا۔  
- 🔧 سیکیورٹی کنٹرولز کو نافذ کرنے کے لیے استعمال ہونے والے ٹولز کی کچھ مثالیں دینا۔  

یہ کورس کیا شامل نہیں کرتا 🙅‍♂️  

- 🚫 مخصوص سیکیورٹی ٹولز کا استعمال کیسے کریں۔  
- 🚫 "ہیکنگ" یا ریڈ ٹیمنگ/جارحانہ سیکیورٹی کیسے کریں۔  
- 🚫 مخصوص کمپلائنس معیارات کے بارے میں سیکھنا۔  

[![ویڈیو دیکھیں](../../translated_images/intro_placeholder.f42382df518f233a1ea3cb1c82ae8f92732bc3ac4ac2b3138cb561d24ca91df5.ur.png)](https://learn-video.azurefd.net/vod/player?id=a0fe1cef-c064-4d59-97a9-e89e12a99b4d)  

جب آپ اس کورس کو مکمل کر لیں، تو آپ ہمارے Microsoft Learn ماڈیولز میں سے کچھ پر منتقل ہو سکتے ہیں۔ ہم تجویز کرتے ہیں کہ آپ اپنی تعلیم کو [Microsoft Security, Compliance, and Identity Fundamentals](https://learn.microsoft.com/training/paths/describe-concepts-of-security-compliance-identity/?WT.mc_id=academic-96948-sayoung) کے ساتھ جاری رکھیں۔  

آخر میں، آپ [Exam SC-900: Microsoft Security, Compliance, and Identity Fundamentals exam](https://learn.microsoft.com/credentials/certifications/exams/sc-900/?WT.mc_id=academic-96948-sayoung) دینے پر غور کر سکتے ہیں۔  

> 💁 اگر آپ کے پاس اس کورس کے بارے میں کوئی رائے یا تجاویز ہیں اور کوئی مواد جو ہم نے شامل نہیں کیا، تو ہمیں آپ کی رائے سن کر خوشی ہوگی!
