In deze snel veranderende tijd waarin AI-technologie wordt omarmd, is het nog belangrijker om te begrijpen hoe IT-systemen beveiligd kunnen worden. Deze cursus is ontworpen om je de basisprincipes van cyberbeveiliging te leren en je leertraject op het gebied van beveiliging te starten. Elke les bevat een korte quiz en links naar aanvullende literatuur als je dieper in het onderwerp wilt duiken.
